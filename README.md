# BaseGrid

BaseGrid is a decentralized compute network that connects AI/ML workloads with distributed GPU nodes. This repository contains the complete stack for the BaseGrid platform:

- **Frontend**: Next.js application for users to submit tasks and view leaderboards.
- **Backend**: NestJS REST API for off-chain profile and task metadata storage.
- **Contracts**: Hardhat project with Solidity smart contracts for Base network integration.
